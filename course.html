Wireless Communications and Applications

  • Course description: This course is given to graduate students. The main content of this course includes fundemetal theory of wireless communications (signal processing and communications system), overview of cellular and wifi systems and networking, emerging applications including wireless power transfer, wireless powered communications, low-power backscatter communications (e.g., RFID and ambient backscatter), wireless indoor positioning (e.g., Wi-Fi- and light-based positioning). This course also gives a brief introduction on programming and simulation skills for wireless networks. Group project counts a very large portion in this course.

  • Classes: 2015 Fall, 2016 Fall, 2017 Fall, 2018 Spring
